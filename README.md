- Docker & Docker Compose
- Java 23+
- Maven 3.8+

1. Clone the repository:
   ```bash
   git clone 
2. Go to the directory:
   ```bash
   cd dream-tournament

3. Docker:
   ```bash
   docker-compose up --build
